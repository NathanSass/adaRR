A quest to automate the life of a junior architect - forever laying out restrooms.

Click [Here](https://young-dusk-3643.herokuapp.com/) to see the work in progress

![HI](http://i.imgur.com/ghuCaLz.gif)

##### Getting Started

Load dependencies
```
npm install
```

Start server
```
npm start
```

Start developing
```
gulp default
```
